<altIdentifier>

<altIdentifier> (alternative identifier) contains an alternative or former structured identifier used for a manuscript, such as a former catalogue number. [10.4 The Manuscript Identifier]
Modulemsdescription — Manuscript Description
Attributesatt.global (@xml:id, @n, @xml:lang, @xml:base, @xml:space) (att.global.rendition (@rend, @style, @rendition)) (att.global.linking (@corresp, @synch, @sameAs, @copyOf, @next, @prev, @exclude, @select)) (att.global.analytic (@ana)) (att.global.facs (@facs)) (att.global.change (@change)) (att.global.responsibility (@cert, @resp)) att.typed (@type, @subtype)
Contained by
msdescription: msIdentifier msPart
May contain
core: note
header: idno
msdescription: collection institution repository
namesdates: bloc country district geogName placeName region settlement

   att.global.rendition.attributes,
   att.global.linking.attributes,

      institution?,
      repository?,

      note?
   )
}
Example
<altIdentifier>
 <settlement>San Marino</settlement>
 <repository>Huntington Library</repository>
 <idno>MS.El.26.C.9</idno>
</altIdentifier>
Show all 
Note

An identifying number of some kind must be supplied if known; if it is not known, this should be stated.
